# 📊 Student Performance Analysis

## 📖 Project Overview

This project presents an exploratory data analysis (EDA) of the **Student Performance** dataset (`student-mat.csv`) using Python. The analysis includes data loading, cleaning, statistical exploration, and data visualization to identify factors that may influence students' academic performance.

---

## 🎯 Objectives

The project aims to:

* 📥 Load the dataset using **Pandas**
* 🧹 Perform data inspection and cleaning
* 📈 Analyze student performance using descriptive statistics
* 🔍 Explore the relationship between study time and final grades
* 👥 Compare academic performance across genders
* 📊 Visualize key findings using charts
* 📝 Document each step with Markdown in a Jupyter Notebook

---

## 📂 Dataset

* **File:** `student-mat.csv`
* **Format:** CSV
* **Domain:** Student Academic Performance

---

## ✅ Tasks Performed

### 📥 1. Data Loading

* Imported the dataset using **Pandas**
* Verified successful data loading

### 🧹 2. Data Exploration & Cleaning

* Checked for missing values
* Removed duplicate records
* Inspected dataset shape, columns, and data types

### 📊 3. Data Analysis

Performed the following analyses:

* 📌 Calculated the average final grade (`G3`)
* 📌 Counted students scoring above **15**
* 📌 Evaluated the correlation between **study time** and **final grade**
* 📌 Compared average final grades between male and female students

### 📈 4. Data Visualization

Generated the following visualizations:

* 📉 Histogram of final grades
* 📍 Scatter plot: Study Time vs Final Grade
* 📊 Bar chart: Average Final Grade by Gender

### 📝 5. Documentation

The notebook includes Markdown explanations describing:

* Data loading
* Data cleaning
* Analysis process
* Visualizations
* Key observations and conclusions

## ▶️ Usage

1. Download the `student-mat.csv` dataset.
2. Place it in the project directory.
3. Launch Jupyter Notebook:

jupyter notebook

4. Open `Student_Performance_Analysis.ipynb`.
5. Run all cells to reproduce the analysis and visualizations.
